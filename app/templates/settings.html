Infrastructure Settings

The following security settings are environment-variable and config-file only and cannot be changed here:

  • auth.secure_cookies — whether auth cookies use the Secure flag (also DECIDERO_SECURE_COOKIES env var)
  • JWT_SECRET_KEY — token signing key (env var only)
  • DECIDERO_ENCRYPTION_KEY — data encryption key (env var only)
