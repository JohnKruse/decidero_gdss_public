# AI Meeting Designer - Product Requirements Document

## Overview
Build an AI-driven Meeting Designer for the Decidero GDSS platform that acts as an automated Collaboration Engineer. Through a conversational interface, it elicits meeting goals, group dynamics, and constraints from the facilitator, then generates a research-grounded agenda of sequenced collaborative activities with explanations for every design choice. The generated agenda pre-fills the existing Create Meeting page via sessionStorage handoff.

## Architecture
- Backend: Python FastAPI, SQLite, Jinja2 templates
- Frontend: Vanilla JavaScript (no framework), custom CSS design system
- AI: Multi-provider support (Anthropic, OpenAI, Azure, Google, custom endpoints) via vendor-agnostic client
- Integration: Extends existing plugin system (ActivityPluginManifest), reuses existing meeting creation flow
